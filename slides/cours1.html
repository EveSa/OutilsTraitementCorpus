## 🛠Focus sur les tâches🛠 ## 🔎Où trouver des corpus🔍 -vertical- ### 📄📚Qu'est ce qu'un corpus📚📄 ```js[1|2|3] Une collection de documents Sur un thème particulier Pour une tâche particulière ``` --- Avant de revenir sur ces caractéristiques : -vertical- ### ❓ A quoi ça sert les data ❓ **Pour l'apprentissage** à partir des observations (*données*), on peut déterminer des paramètres (*features*) qui vont nous permettre de prédire le futur -vertical- **un exemple** ⛅Prévoir la météo⛅: - la saison ❄🌺 - la couverture nuageuse ☁ - le vent 🍃 - la météo des régions voisines 🌍 - ... -vertical-

🤖Pour les machines, c'est pareil🤖

On a besoin de **données** pour apprendre les **paramètres** qui permettent de prédire le futur → On collecte des datasets C'est comme ça qu'on obtient des modèles **prédictifs** -vertical- ## On peut maintenant revenir sur les caractéristiques de notre corpus📚📄 -vertical- ### Une collection = un nombre représentatif de données **Pour couvrir suffisemment de cas** --- ⛅Prévoir la météo⛅ - nuage ☁ = pluie 🌧 - vent 🍃 = beau temps ☀ - nuage ☁ + vent 🍃 = ❓ -vertical- ### Un thème **Pour éviter le bruit** --- ⛅Prévoir la météo⛅ - mon humeur de ce matin 🎭 - le nombre de café que j'ai bu 🍵 - les traces de bouctins devant chez moi 👣 - ... -vertical- ### Pour une tâche donnée **Pour avoir la sortie voulue** --- | | | |:---:|:---:| | ❄🌺 | ❄🌺 | | ☁ | ☁ | | ... | ... | |→ |→ | |⛅Prédire la météo⛅|🦜Parler de la pluie et du beau temps🦜| -vertical- ### ❓ A quel point c'est important ❓ ❤C'est le nerf de la guerre❤ Les entreprises publient : ✔ 🏠 les architectures ✔ 🔩 les poids des modèles ❌ 📚 mais pas les data -vertical- ### Obtenir des données → Avec internet, c'est plus si compliqué ### Obtenir des données *utiles* → C'est très couteux 💸💸💸 -vertical- ### Les problématiques des données - La propreté des données ✨ - quand on crawl les data sur internet, on a de tout - Le RGPD (pour les données confidentielles🕵️‍♂️) - particulièrement visible dans le domaine médical → alors que les modèles sont très bons pour prédire les pathologies - L'annotation des données 🏷 - Pour réaliser certaines tâches, on a besoin d'experts et ça coûte cher -horizontal- ## 🛠Les tâches et les corpus🛠 En *Machine Learning*, on fait de la prédiction : - pour catégoriser les éléments existants - pour découvrir de nouveaux éléments -vertical-
